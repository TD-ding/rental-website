# 前端文档

## 页面结构

| 页面 | 文件 | 功能 |
|------|------|------|
| 首页 | `public/index.html` | 房源列表、搜索筛选、排序、收藏 |
| 详情页 | `public/detail.html` | 房源详情、联系方式、预约表单 |
| 登录页 | `public/login.html` | 管理员登录 |
| 管理后台 | `public/admin.html` | 房源管理、预约管理、用户列表 |

## 核心功能

### 房源搜索
- 关键词搜索（标题、地址、描述）
- 价格区间筛选
- 户型筛选
- 排序：最新发布 / 价格升序 / 价格降序
- 搜索后自动平滑滚动到结果区域

### 收藏功能
- 基于 localStorage 存储，刷新不丢失
- 首页卡片上的 ♡ 按钮收藏/取消
- 详情页收藏按钮
- 导航栏"我的收藏"展开收藏面板

### 预约看房
- 详情页底部预约表单
- 前端验证：姓名、手机号格式、日期
- 提交按钮防重复点击

### 管理后台
- Cookie + 服务端 Session 认证
- Tab 切换：房源管理 / 预约管理 / 用户列表
- 统计卡片：房源总数、待处理/已确认预约、用户数
- 房源增删改查（模态框表单）
- 预约状态处理（确认/拒绝）
- Session 过期自动跳转登录页

## 数据流

- `fetchJSON()` 封装所有 API 调用，自动携带 cookie（`credentials: 'include'`）
- 所有动态内容通过 `esc()` 函数转义防 XSS
- 错误统一通过 `toast()` 提示

## 安全措施

- XSS 防护：所有动态内容通过 `esc()` 转义
- 输入验证：前端逐字段验证 + 后端二次校验
- 按钮防重复提交：提交时 disable + 文字变更
