# 管理后台文档

## 访问

管理后台地址：`/admin.html`，需要先通过 `/login.html` 登录。

默认管理员：`admin` / `admin123`

## 功能

### 统计概览
页面顶部展示 4 个统计卡片：
- 房源总数
- 待处理预约数
- 已确认预约数
- 用户总数

### 房源管理
- 查看所有房源列表
- 添加房源（点击"+ 添加房源"按钮）
- 编辑房源（点击"编辑"按钮）
- 删除房源（点击"删除"按钮，确认后删除）

### 预约管理
- 查看所有预约列表，含房源标题、预约人信息、状态
- 待处理预约可"确认"或"拒绝"
- 状态标签：待处理（黄色）、已确认（绿色）、已拒绝（红色）

### 用户列表
- 查看所有注册用户

## 权限

- 管理后台所有操作需要登录
- Session 过期自动跳转登录页
- 退出登录清除服务端 Session 和客户端 Cookie
