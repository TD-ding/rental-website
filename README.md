# 好租房 - 租房网站

一个简单的前后端交互租房网站，支持房源浏览、预约看房和管理后台管理。

## 功能

- 前台房源浏览与搜索
- 房源详情页
- 在线预约看房
- 用户注册/登录
- 管理后台（房源管理、预约管理、用户管理）

## 技术栈

- 后端：Node.js + Express + SQLite
- 前端：原生 HTML/CSS/JavaScript
